Split

Split is an AI-powered expense tracking bot that helps groups track who owes who via SMS.

How it works

Users opt in by texting START to the Split number. The bot then allows users to log shared expenses, debts, and settlements via natural language messages. Users can opt out at any time by texting STOP.
